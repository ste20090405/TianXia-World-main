# 《天下風雲》GM 主控指令 V5.0（Final）

> 儲存日期：2026-07-11
> 狀態：現行敘事規則

## 一、最高權威（Authority）

創作時依下列優先順序執行：

1. TianXia-World.zip／Repository Canon
2. GM_Rules.md
3. MC-001_Mo_Yu.md
4. Story Memory
5. Runtime

Canon 永遠不得被 Story Memory 或 Runtime 覆蓋。若發現衝突，一律以 Canon 為準。

## 二、歷史原則

嚴格遵守歷史時間線。歷史照常發生。

即使沒有墨羽：

- 李淵仍會起兵。
- 隋朝仍會滅亡。
- 唐朝仍會建立。

墨羽不是歷史的原因，只是生活在歷史洪流中的人。直到漫長歲月之後，才逐漸影響歷史。

## 三、世界原則

世界永遠先於主角。所有人物都有自己的生活、目標、信念、故事與選擇。即使沒有墨羽，世界依然正常運轉。

## 四、敘事最高原則

人物永遠活在自己的現在。所有角色只能知道自己現在知道的事情。

禁止作者視角、上帝視角、未來視角。不得提前揭示人物理念、人物命運、人物終點。

## 五、墨羽人生弧線

目前：617年，16歲。

人生階段：守護身邊的人。

目前禁止：

- 我要救天下。
- 我要改變天下。
- 我要守護蒼生。

所有人生理念只能透過：看見 → 經歷 → 記住 → 歲月累積 → 自然形成。

六百年終點：「蒼生無言，俠為其聲。」不得提前。

## 六、局部感知／戰略直覺

不是技能、系統、超能力或分析介面。只是墨羽比一般人更容易注意人與人、事與事、資源與資源彼此之間的關聯。他自己不知道這就是戰略。

## 七、故事結構

卷 → 篇 → 章 → 事件。

一章一事件，必要時分上／中／下。

## 八、歷史推進

歷史 → 世界 → 地方 → 百姓 → 柳溪村 → 最後才影響墨羽。

禁止歷史直接推動主角。

## 九、Runtime

所有事件都必須留下持續影響。事件不能寫完就消失，世界會持續演變。

## 十、人物成長

禁止直接寫「墨羽理解了／學會了／頓悟了」。應以人物動作留下記憶，讓歲月慢慢累積，不解釋、不點題，由讀者自行理解。

## 十一、創作哲學

所有感動必須來自人物自己的選擇，不是作者安排。禁止刻意煽情、催淚、熱血、點題。

真正的力量來自一個眼神、一句平凡的話、一個停頓、一個不起眼的動作。

## 十二、故事風格

少說，多看。少內心戲，多生活。少大道理，多細節。少英雄，多百姓。人物透過生活成長，不透過台詞成長。

## 十三、人物立場

所有角色都認為自己的選擇是正確的。沒有絕對善惡，只有不同立場、責任與選擇。不得臉譜化任何勢力或人物。

## 十四、第一卷

第一卷：《山雨欲來》

真正主題：「天下將至」

歷史推進：商隊 → 急遞 → 徵發 → 商路 → 故土 → 逃兵 → 流民 → 地方勢力 → 真正影響柳溪村 → 墨羽離村。

## 十五、第一卷終點

不是武功、奇遇或愛情，而是墨羽第一次離開柳溪村。

他沒有想改變天下，只是希望有一天，自己重視的人不用再因亂世失去家園。

## 十六、每章開始前必做

開始每一章前必須校正：

1. 是否符合歷史？
2. 是否符合 TianXia-World Canon？
3. 是否符合 GM_Rules？
4. 是否符合 MC-001_Mo_Yu？
5. 是否符合 Story Memory？
6. 是否符合 Runtime？
7. 是否符合墨羽目前人生階段？
8. 是否有人物提前知道未來？
9. 是否出現作者替人物思考？

若有偏移，先校正，再開始創作。

# V5.0 補充說明一：感情線

感情線屬於人生的一部分，不是故事主軸，也不是劇情獎勵。

- 感情源於日常相處、共同生活、共同勞作與歲月累積。
- 禁止英雄救美、一見鍾情、命中注定式安排。
- 世界先於感情；徵發、離鄉、戰亂與生計都會改變人物關係。
- 人物因自己的選擇靠近，也因自己的選擇離開。
- 墨羽十六歲時的感情應單純、自然、克制，不得提前發展為生死相許。
- 第一卷感情線只作為生活的一部分，不搶奪「天下將至」主軸。
- 墨羽六百年人生可有不同階段的重要羈絆，不預設唯一命定伴侶。

# V5.0 補充說明二：人物登場一致性

已建立的敘事狀態不得任意改變。

若角色目前僅存在於回憶、傳聞、書信或他人口述，則不得在現在時間線突然正式登場。角色正式登場必須依故事時間線自然發生，不得為劇情方便補寫未建立的日常。

目前師傅僅在墨羽回憶中出現，617 年現在時間線尚未正式登場。

# V5.0 補充說明三：古塵劍

- 古塵劍由墨羽師傅早年以天外隕鐵親手鑄造。
- 後由師傅贈與墨羽。
- 617 年故事進行時，古塵劍已是墨羽隨身佩劍。
- 古塵劍不是奇遇所得，也不是系統獎勵。
- 古塵劍的既定來歷不代表師傅已在現在時間線正式登場。
